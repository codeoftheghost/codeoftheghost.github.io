Persiapan Memulai Pemrograman Python#

Sebelum mulai melakukan pemrograman dengan bahasa python, kita perlu untuk menginstall beberapa perangkat lunak atau software. Perangkat lunak ini akan digunakan selama kita menulis kode program python dan menjalankannya. Berikut beberapa perangkat lunak yang harus kita pasang:

Kode Editor#

Kode editor adalah perangkat lunak yang dirancang untuk menulis, mengedit, dan mengelola kode program dalam berbagai bahasa pemrograman, termaksud python. Perangkat lunak ini dapat membantu mempermudah kita untuk menulis kode program dengan berbagai fitur-fitur yang disematkan didalamnya seperti, penyorotan sintaks, indentasi otomatis, pemeriksaan kesalahan penulisan, dan lain-lain.

Ada berbagai kode editor yang dapat kita gunakan, diantaranya yang paling banyak digunakan ialah Visual Studio Code atau sering disingkat VSCode, Zed, Sublime Text, Atom, dan masih banyak lagi. Pada catatan ini saya menggunakan VSCode, tetapi jangan khawatir jika kalian ingin menggunakan kode editor yang lain. Karena kalian tetap bisa mengikuti catatan ini dengan menggunakan kode editor lain tanpa ada masalah.

Menginstall Kode Editor VSCode di Linux#

Pada distribusi linux, kita bisa menginstall VSCode menggunakan dua cara. Cara pertama ialah menggunakan Package Manager bawaan dari distribusi linux yang kita gunakan. Cara ini adalah yang paling gampang menurut saya. Silahkan buka terminal, kemudian ketikkan perintah dibawah ini lalu tekan Enter:


sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc

echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" | sudo tee /etc/yum.repos.d/vscode.repo > /dev/null

sudo dnf update

sudo dnf install code


sudo apt-get install wget gpg

wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg

sudo install -D -o root -g root -m 644 packages.microsoft.gpg /etc/apt/keyrings/packages.microsoft.gpg

echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" |sudo tee /etc/apt/sources.list.d/vscode.list > /dev/null

rm -f packages.microsoft.gpg

sudo apt install apt-transport-https

sudo apt update

sudo apt install code

Perintah diatas menggunakan sudo, yang berarti kemungkinan besar kalian akan diminta untuk memberikan password. Silahkan isikan password kalian, kemudian tekan Enter lagi.

Jika kalian menggunakan distribusi selain dari distribusi yang sudah saya sebutkan diatas, silahkan dibaca dokumentasi resmi VSCode pada bagian instalasi di halaman code.visualstudio.com/docs/setup/linux.

Cara kedua ialah dengan mengunduh file binary installer VSCode dari halaman berikut, code.visualstudio.com/Download.

Dibagian linux, silahkan pilih yang sesuai dengan format package .deb atau .rpm distribusi kalian, dan arsitektur perangkat kalian x64, Arm32, atau Arm64.

Jika proses download file binary installer VSCode telah selesai, silahkan double klik kiri pada file tersebut untuk menjalankan proses instalasinya. Kita akan diminta untuk memasukkan password, silahkan masukkan kemudian tekan Enter. Proses instalasi selesai ditandai dengan file VSCode yang akan muncul di menu aplikasi, atau shortcut di desktop kita.

Interpreter Python#

Tidak seperti bahasa pemrograman yang dikompilasi seperti C/C++, Java, Go, dan lain-lain, dimana kode program yang kita tulis, akan langsung diterjemahkan menjadi instruksi mesin sekali saat proses kompilasi, dan menghasilkan file binary yang siap dijalankan. Di python, kita membutuhkan perangkat lunak interpreter untuk menjalankan kode program yang kita tulis. Interpreter ini akan menerjemahkan kode program kita menjadi instruksi mesin, dan mengeksekusinya secara langusng, baris demi baris.

Menginstall Interpreter Python di Linux#

Pada distribusi linux, umumnya interpreter python sudah terinstall secara default. Jadi kita tidak perlu lagi menginstall-nya. Untuk memastikan apakah interpreter python sudah terinstall pada distribusi yang kita gunakan, bisa dengan mengetikkan perintah dibawah ini pada terminal kita: 


python --version

Perintah diatas sebenarnya berguna untuk mengecek versi dari interpreter python yang terinstall. Jika interpreter python sudah terinstall, maka akan ada output berupa versi yang terpasang seperti dibawah ini: 


Python 3.12.7

Output diatas menunjukkan bahwa python terinstall dengan versi Python 3.12.7. Jika output berupa pesan error atau sejenisnya seperti dibawah ini kemudian tekan Enter: 


bash: python: command not found...

Ini berarti interpreter python belum terinstall. Kita bisa menginstallnya dengan cara membuka terminal, kemudian ketikkan perintah dibawah ini: 


sudo dnf install python



sudo apt update && sudo apt install python3 -y

Perintah diatas menggunakan sudo, yang berarti kemungkinan besar kalian akan diminta untuk memberikan password. Silahkan isikan password kalian, kemudian tekan Enter lagi.

Jika kalian menggunakan distribusi linux selain dari contoh diatas, silahkan sesuaikan dengan perintah penginstall-an dari package manager yang digunakan.
